Welcome to vLab's documentation!
================================

.. image:: vlab_icon.png
    :width: 150px
    :height: 150px
    :align: right


vLab is an application that provides self-service virtual computer labs.
With vLab you can do all kinds of things like:

- Create OneFS cluster, and other EMC products
- Setup crazy networks with WAN emulators
- Gain *hands-on* experience with the products we support
- *and so much more!*

If you're **new**, then you'll want to checkout the **Getting Started** page first.
